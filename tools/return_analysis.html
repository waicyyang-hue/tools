📖 使用说明
1. 下载FBA退货报告
路径：卖家后台 → 订单 → 退货与价值回收 → 洞察与机遇 → 资源中心 → 亚马逊物流退货报告 → 下载
建议日期范围：最近12个月
格式：CSV
2. 上传并查询
上传CSV文件 → 点击"查看退货排名"查看退货率最高的ASIN
输入需要查询的子ASIN → 点击"开始分析"查看详细退货原因
查询父ASIN时，输入多个子ASIN，用逗号或空格分隔
3. 如有问题，请联系 cyangb@
